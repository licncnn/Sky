## 开始使用
1. Star 本项目；（当然你不Star我也查不到
2. 在 [Releases](https://github.com/isakurai/BigBabol/releases) 页面下载最新发行包；
3. 将主题上传到 `/usr/themes/` 目录下并解压；
4. 也许需要修改文件夹名称为**Sky**并删除压缩包
5. 登录到 Typecho 的后台，在 控制台 -> 外观 启用主题
